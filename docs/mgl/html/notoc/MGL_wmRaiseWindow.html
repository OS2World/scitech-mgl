MGL_wmRaiseWindow

Raises the window.

Declaration

void MGLAPI MGL_wmRaiseWindow(
    window_t *wnd)

Prototype In

mgraph.h

Description

Raises the window in front of its siblings. No part of the window is covered by any of its siblings after this operation.

MGL_wmSetWindowFlags, MGL_wmLowerWindow
